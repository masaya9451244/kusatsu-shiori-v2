## プロジェクト概要

## 3. 各セクション詳細

### A. 選択画面（初期表示）
- **レイアウト**: 中央に2つのボタン
- **背景**: Onsen White → Brick Onsenへの縦グラデーション
- **装飾**: shigureni温泉系イラストを薄く配置
- **ボタン**:
  - 「✨ 初めて見る」（サプライズ演出へ）
  - 「📖 もう一度見る」（本編へ直接）

### B. サプライズ演出（初回のみ）

**ステップ1: お祝いメッセージ**
- テキスト: 「🎂 菜々穂さん、お誕生日おめでとう！ 🎂」
- 紙吹雪（コンフェッティ）がふわっと舞い落ちる
- 文字が1文字ずつ優しくフェードイン
- 背景: 柔らかいピンク〜煉瓦色のグラデーション
- 「次へ →」ボタンで進む

**ステップ2: 封筒演出（行き先ティザー）**
- 画面中央に封筒のイラスト
- テキスト: 「特別な招待状が届いています」
- 「タップして開ける」のガイド
- タップすると封筒が開くアニメーション
- 中から手紙が出てきて「今回行く旅行の行き先を発表します。」

**ステップ3: スクラッチカード（行き先発表）**
- スクラッチカード風のエリアを表示
- テキスト: 「★ ここをこすってね ★」
- 指でなぞるとキラキラしながら削れる
- 下から「草津温泉！」の文字が現れる
- 全部削ると湯気エフェクト＆お祝い演出

**ステップ4: 宿ティザー**
- テキスト: 「そしてそして、宿泊する宿は！」
- わくわく感を煽るバウンスアニメーション

**ステップ5: 宿の魅力スライドショー**
写真と説明文を順番にフェードで切り替え:
1. 外観 → 「洋館スタイルの欧風リゾートホテル」「ドイツ・グリム童話のテイストを取り入れたお部屋」
2. 星空テラス・露天風呂 → 「満天の星を眺める露天風呂」「貸し切り露天風呂も完備」
3. 温泉 → 「湯畑・万代鉱・湯川の湯 3つの源泉を堪能」
4. 朝食 → 「ステンドグラスが印象的なレストラン」「バイキング形式の朝食」
- 各スライド 3秒程度で自動切り替え or スワイプ

**ステップ6: 宿発表**
- テキスト: 「🌟 ラビスタ草津ヒルズ！ 🌟」
- 星がキラキラと散るエフェクト
- エレガントなフォントで大きく表示

**ステップ7: 締め**
- テキスト: 「素敵な思い出を作ろうね 💕」
- 「しおりを見る →」ボタン
- クリックでメインコンテンツへフェードイン

## 5. アニメーション仕様

### サプライズ演出（初回選択時）

**お祝いメッセージ**
- 紙吹雪（コンフェッティ）: Canvas or CSS animationで実装
- 文字フェードイン: 1文字ずつ0.1秒間隔

**封筒演出**
- 封筒: CSSアニメーションで開封（蓋がパカッと開く）
- 手紙: 封筒の中からスライドアップ

**スクラッチカード**
- Canvas API でスクラッチ実装
- タッチ/マウスでなぞった部分が消える
- キラキラパーティクル: 削った箇所から発生
- 70%以上削ると自動で全表示

**行き先発表（草津温泉）**
- 文字が拡大しながらフェードイン（scale: 0.5 → 1）
- 湯気パーティクル: ふわふわ上昇

**宿スライドショー**
- 写真: クロスフェード（opacity切り替え）
- テキスト: 下からスライドイン
- 自動進行: 3秒間隔

**宿発表（ラビスタ草津ヒルズ）**
- 星のエフェクト: キラキラが周囲に散る
- 文字: エレガントなスケールアップ

### メインコンテンツ表示時
- 各セクションが0.5秒ディレイで順次フェードイン

### スクロール時
- shigureniイラストが3〜4px浮遊
- セクションは縦方向に柔らかくフェードイン
- ボタン: 角丸、ホバーで白い湯気グローが差す
